Create kernal envionment:
=============================

use c:\Arun\PythonEnv to store all virtual environments

![image](https://github.com/user-attachments/assets/45fea65c-04b2-4c0e-8d42-61512033f585)


C:\Arun\PythonEnv>python -m venv DEEP_LEARNING



Activate
=========

C:\Arun\PythonEnv\DEEP_LEARNING\Scripts>activate

Now inside virtual env , you can now install all needed packages for deeplearning in a
separate folder without disturbing all  other environment.
(DEEP_LEARNING) C:\Arun\PythonEnv\DEEP_LEARNING\Scripts>
 

install ipkernel & jupyter to list in vscode:
==============================================

if needed reinstall
   
 C:\Arun\PythonEnv\DEEP_LEARNING>   python.exe -m pip install ipykernel -U --force-reinstall
